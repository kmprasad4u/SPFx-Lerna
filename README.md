## SPFx-Lerna

### Building the code
* Change Site Url in src\SPFx-WebPart\Config\serve.json
* In Termainal 1
    * cd src\SPFx-Library
    * npm install
    * gulp serve
* In Terminal 2
    * cd src\SPFx-WebPart
    * npm install
    * gulp serve
* Edit the page (NOT online workbench) and add Sample WebPart
